# IIA README

## iPad Illustration App (Private Use)

iPad専用の個人用お絵描きアプリです。
Apple Pencil対応のキャンバス、レイヤ、ブラシ設定など、一般的なイラスト制作に必要な機能を自分用に実装することを目的としています。
App Store などでの公開予定はなく、**開発者本人だけが使うことを前提**としています。

---

## ✨ 機能概要（予定含む）

* Apple Pencil 対応のキャンバス（PencilKit ベース）
* ブラシツール / 消しゴム
* Undo / Redo
* カラーピッカー
* レイヤ機能（追加 / 削除 / 表示非表示 / 順序入れ替え）
* レイヤごとの不透明度
* ブラシ設定（太さ / 不透明度）
* ブラシテクスチャ（将来対応）
* レイヤのブレンドモード（将来対応）
* 画像書き出し（PNG / JPEG など）
* ドキュメントの保存・読み込み（ローカルのみ）

> このアプリは個人利用専用であり、第三者への配布やストア公開を前提としていません。

## 🧑‍💻 開発環境セットアップ

1. macOS に最新の **Xcode** をインストールする
2. iPad を Mac に接続し、開発用デバイスとして登録する（必要であれば Developer Mode を有効化）
3. 本プロジェクトを Xcode で開く
4. 自身の Apple ID を利用した無料プロビジョニングで署名設定を行う
5. ターゲットデバイスとして iPad を選択し、Run（▶）してビルド・インストール

### 無料アカウントでの制限

* アプリは一定期間（数日〜1週間程度）で署名の有効期限が切れるため、

  * 必要に応じて Xcode から再ビルド・再インストールする
* App Store や TestFlight での配信は行わない

個人用としてはこの運用で問題ない想定です。

## 🚧 ロードマップ

### フェーズ1：最小限のお絵描きアプリ

* PencilKit を使ったキャンバス
* 単一または少数レイヤ対応
* ブラシ/消しゴム/色変更/Undo/Redo
* ドキュメント保存・読み込み

### フェーズ2：描き心地と表現力の向上

* ブラシ設定の拡張（ブラシプリセット、手ぶれ補正 など）
* ブラシテクスチャ（鉛筆/ペン/水彩調など）
* レイヤのブレンドモード
* 高解像度キャンバスでのパフォーマンス検証

### フェーズ3：拡張要素（必要になったら）

* Metal を用いた独自レンダリングの検討
* キャンバステクスチャ（紙の質感など）
* 簡易フィルタ（ぼかし、トーン調整など）
